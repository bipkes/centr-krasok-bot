## 📌 О проекте

Тестовое задание: разработать Telegram-бота с AI-ассистентом, который отвечает на вопросы о компании «Центр Красок #1».

### Задачи которые решает бот:
- Отвечает на вопросы клиентов 24/7 без участия менеджера
- Собирает заявки и передаёт менеджеру мгновенно
- Общается на языке пользователя (рус/каз/eng)
- Направляет клиента к специалисту если не знает ответа

---

## ⚙️ Стек технологий

| Инструмент | Назначение |
|-----------|-----------|
| **n8n Cloud** | Автоматизация и оркестрация workflow без кода |
| **OpenAI GPT-4o-mini** | AI модель для генерации ответов |
| **Telegram Bot API** | Интерфейс общения с пользователем |
| **Google Sheets** | Хранение лидов и заявок клиентов |

### Почему GPT-4o-mini?
- Оптимальное соотношение цена/качество (~$0.05 за 1000 сообщений)
- Достаточно умный для Q&A задач
- Быстрый отклик

### Почему n8n?
- Визуальный конструктор — быстрая разработка MVP
- Встроенные интеграции с Telegram, OpenAI, Google Sheets
- Простой деплой через cloud версию

## 🗂️ База знаний

Бот отвечает строго на основе структурированного системного промпта:

### Что включено:
- ✅ О компании (юр. данные, история, позиционирование)
- ✅ Полный каталог категорий товаров
- ✅ Список 40+ брендов с описанием
- ✅ Все услуги (колеровка, доставка, программы лояльности)
- ✅ Контакты и адреса шоурумов (Алматы x2, Астана)
- ✅ Города доставки по Казахстану
- ✅ Соцсети и мессенджеры
- ✅ Типичные вопросы и готовые ответы

### Защита от галлюцинаций:
- Бот отвечает ТОЛЬКО на основе базы знаний
- Не придумывает цены и акции
- При незнании — предлагает звонок специалиста
- Не обсуждает темы вне компании
